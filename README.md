Battle Blaster is a simple arcade-style third-person tank shooter built in Unreal Engine using C++.  
This project focuses on core gameplay systems and understanding Unreal Engine’s C++ framework.


---

## 📌 Overview

In this project, I developed a basic third-person shooter experience where the player controls a tank and engages in combat using projectile-based attacks.

The goal of this project was to learn and apply fundamental Unreal Engine concepts such as player control, game flow, and gameplay mechanics using C++.

---

## 🧠 Key Features

### 🚗 Player-Controlled Tank
- Modified a third-person character into a tank-style controller
- Implemented movement and rotation logic

### 💣 Projectile System
- Created a projectile-based shooting mechanic
- Handles spawning, movement, and collision

### 🏆 Game Logic & Win Condition
- Implemented basic win condition system
- Structured gameplay flow using GameMode

### 🎮 Player Controller & GameMode
- Extended Unreal’s PlayerController and GameMode classes
- Managed input handling and overall game rules

---

## 🛠️ Tech Stack

- Unreal Engine (C++)
- Gameplay Framework (GameMode, PlayerController)
- Basic physics & collision system

## 🚀 Getting Started

### Requirements
- Unreal Engine (version 5.6 recommended)
- Visual Studio (for C++ development)

### Steps
1. Clone the repository
2. Open the `.uproject` file
3. Build the project if prompted
4. Press Play in the Unreal Editor
